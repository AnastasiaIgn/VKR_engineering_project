**Тема ВКР:** Методы и средства управления инженерными проектами с применением больших языковых моделей

**Автор:** Игнатьева Анастасия Юрьевна

---

## Описание проекта

Программный комплекс для автоматической обработки, векторизации и семантического анализа проектной документации (договоры, проектная документация, сметы). Система обеспечивает:

- Семантический поиск по документам с фильтрацией по источникам
- Выявление смысловых связей между фрагментами документов
- Анализ рисков и несоответствий на основе выявленных противоречий
- Веб-интерфейс для взаимодействия с пользователем

## Установка и запуск

### 1. Требования

- Python 3.9+
- 4 ГБ оперативной памяти (рекомендуется 8 ГБ)

- 
# Создание окружения
python -m venv venv_sbert

# Активация (Windows)
venv_sbert\Scripts\activate

# Активация (Linux/Mac)
source venv_sbert/bin/activate

# Установка зависимостей
pip install -r requirements_clean.txt

## Последовательный запуск модулей

# Шаг 1. Обработка документов (извлечение текста, очистка, чанкинг)
python reprocess_fixed.py

# Шаг 2. Создание эмбеддингов с помощью SBERT (2-3 минуты)
python create_embeddings_sbert.py

# Шаг 3. Построение векторного хранилища Chroma DB
python create_vector_store.py

# Шаг 4. Выявление семантических связей между документами
python extract_semantic_links.py

# Шаг 5. Анализ рисков и формирование отчета
python analyze_risks.py

# Шаг 6. Запуск веб-интерфейса
streamlit run app.py


# Структура проекта

```
VKR_clean/
│
├── docs/                              # Исходные документы
│   ├── proect.pdf                     # Проектная документация
│   ├── proect_pz.pdf                  # Пояснительная записка
│   └── smeta.pdf                      # Сметная документация
│
├── data/                              # Данные (создаются автоматически)

├── reprocess_fixed.py                 # Модуль обработки и чанкинга
├── create_embeddings_sbert.py         # Модуль векторизации (SBERT)

├── analyze_risks.py                   # Модуль анализа рисков
├── app.py                             # Веб-интерфейс (Streamlit)
├── requirements_clean.txt             # Зависимости проекта
└── README.md                          # Описание проекта
